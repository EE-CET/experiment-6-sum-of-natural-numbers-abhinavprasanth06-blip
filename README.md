# Experiment 6: Sum of Natural Numbers

## Problem Statement

For an integer $N$, your task is to calculate the sum of the first $N$ natural numbers.

**Task:** You have to create a function `NumberSum()` which takes the integer $N$ as a parameter and returns the sum of the first $N$ natural numbers.

## Input Format

The first line contains an integer $N$.

## Output Format

Print the sum of the first $N$ natural numbers.

### Example 1

**Input:**

```text
5
```

**Output:**

```text
15
```

### Explanation

The sum of the first 5 natural numbers is $1 + 2 + 3 + 4 + 5 = 15$.
